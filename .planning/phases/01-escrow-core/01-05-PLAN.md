```typescript
interface UpdatePayoutCardRequest {
  card_synonym: string;   // YooKassa card synonym from Payout Widget
  card_last4: string;     // last 4 digits for display
}
```

**Flow:**
1. Get session via `getSession()` -- return 401 if no session
2. Validate session role is 'worker' -- return 403 if not
3. Parse body: validate card_synonym (non-empty string), card_last4 (4 digits)
4. Update workers table using supabaseAdmin:
   ```typescript
   const supabase = getServiceClient();
   const { error } = await supabase
     .from('workers')
     .update({
       payout_card: body.card_last4,
       payout_card_synonym: body.card_synonym,
     })
     .eq('phone', session.user_id); // session.user_id is phone for workers
   ```
5. Return `{ success: true }`

**Error handling:**
- 400 for validation errors
- 401 for no session
- 403 for non-worker role
- 500 for DB errors

**Imports:**
