**Request body (POST):**
```typescript
interface CreateEscrowRequest {
  orderId: string;
  subtotal: number;           // base cost of work
  comboDiscount?: number;     // 15% combo discount (0 if none)
  customerPhone: string;      // required for receipt
  customerEmail?: string;     // optional
  returnUrl?: string;         // optional override, defaults to /order/{id}/status
}
```

**Business logic:**
1. Validate request body: orderId required, subtotal > 0, customerPhone required
2. Fetch order via `getOrderById(orderId)` — return 404 if not found
3. Check `order.escrow_status` is '' (empty string) — return 409 "Payment already initiated" if not
4. Calculate fees:
   ```typescript
   const SERVICE_FEE_RATE = 10; // 10%
   const serviceFee = Math.round(subtotal * SERVICE_FEE_RATE / 100 * 100) / 100;
   const comboDiscount = body.comboDiscount ?? 0;
   const total = subtotal + serviceFee - comboDiscount;
   const payoutAmount = subtotal; // executor gets 100% of subtotal
   ```
5. Generate idempotence key: `escrow-${orderId}-${Date.now()}`
6. Determine returnUrl: `body.returnUrl || process.env.NEXT_PUBLIC_APP_URL + '/order/' + orderId + '/status'`
   - `NEXT_PUBLIC_APP_URL` defaults to `'https://podryad.pro'` if not set
7. Call `createEscrowPayment({ amount: total, subtotal, serviceFee, description: 'Заказ ' + orderId, returnUrl, orderId, customerPhone: body.customerPhone, idempotenceKey })`
8. Update order with escrow financial fields:
   ```typescript
   await updateOrder(orderId, {
     subtotal,
     service_fee_rate: SERVICE_FEE_RATE,
     service_fee: serviceFee,
     combo_discount: comboDiscount,
     total,
     payout_amount: payoutAmount,
     yookassa_payment_id: payment.id,
     customer_phone: body.customerPhone,
     customer_email: body.customerEmail || null,
   });
   ```
9. Return `{ confirmationUrl: payment.confirmationUrl, paymentId: payment.id }`

**Error handling:**
- Wrap in try/catch, log errors with `console.error`
- Return `NextResponse.json({ error: message }, { status: code })`
- 400 for validation errors, 404 for order not found, 409 for already-initiated, 500 for YooKassa/DB errors

**Security:**
- This is a server-side route. No session/auth check needed since it's called from the app's own order flow (n8n/Telegram sends order_id).
- The orderId itself is the security token (UUIDs are unguessable).
- Use `getServiceClient()` (service_role) for all DB operations.

**Import pattern:**
```typescript

import { createEscrowPayment } from '@/lib/yukassa';
import { getOrderById, updateOrder } from '@/lib/db';

const YOOKASSA_CIDRS = [
  { ip: 0xB9474C00, mask: 0xFFFFFFE0 }, // 185.71.76.0/27
  { ip: 0xB9474D00, mask: 0xFFFFFFE0 }, // 185.71.77.0/27
  { ip: 0x4D4B9900, mask: 0xFFFFFF80 }, // 77.75.153.0/25
  { ip: 0x4D4B9A80, mask: 0xFFFFFF80 }, // 77.75.154.128/25
  { ip: 0x4D4B9C0B, mask: 0xFFFFFFFF }, // 77.75.156.11/32
  { ip: 0x4D4B9C23, mask: 0xFFFFFFFF }, // 77.75.156.35/32
];

function parseIPv4(ip: string): number {
  const parts = ip.split('.').map(Number);
  return ((parts[0] << 24) | (parts[1] << 16) | (parts[2] << 8) | parts[3]) >>> 0;
}

function isYooKassaIP(ipStr: string): boolean {
  if (process.env.NODE_ENV === 'development') return true; // skip in dev
  try {
    const ip = parseIPv4(ipStr);
    return YOOKASSA_CIDRS.some(c => (ip & c.mask) === (c.ip & c.mask));
  } catch {
    return false;
  }
}
```
