Авторизация заказчика без SMS (бесплатно)

Как заказчик попадает на свой дашборд без платных SMS?

A

Секретная ссылка через мессенджер (рекомендую)

После создания заказа заказчик получает уникальную ссылку в MAX/Telegram/Email:
podryad.pro/my/abc123xyz

Ссылка привязана к его телефону. Кто знает ссылку — видит свои заказы. Можно добавить в закладки.
Если потерял — запрашивает повторно через бота или форму (ввёл телефон → получил ссылку в мессенджер).

Плюсы

  • Полностью бесплатно
  • Ноль трения — один клик из мессенджера
  • Работает через каналы, которые уже есть (MAX/TG/Email)

Минусы

  • Кто получил ссылку — тот имеет доступ (но можно добавить PIN)
B

Телефон + 4-значный PIN-код

Заказчик при первом заказе придумывает PIN (4 цифры). Чтобы войти в дашборд: вводит телефон + PIN.
Забыл PIN → получает новую ссылку-сброс через мессенджер.

Плюсы

  • Бесплатно
  • Безопаснее — нужно знать и телефон, и PIN
  • Привычный формат

Минусы

  • Ещё одно поле в форме заказа
  • Заказчики забывают PIN
C

Код подтверждения через мессенджер (OTP без SMS)

Заказчик вводит телефон на странице входа → получает 6-значный код в MAX/Telegram/Email → вводит код → попадает в дашборд. По сути SMS-код, но через мессенджер.

Плюсы

  • Бесплатно
  • Привычный UX (как SMS-код)
  • Безопасно — код одноразовый

Минусы

  • Два шага: ввести телефон → переключиться в мессенджер → скопировать код → вернуться
  • Нужно, чтобы бот уже знал пользователя
