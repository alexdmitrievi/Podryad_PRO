Админ-панель

/admin
Панель управления
Обновлённые табы:
  • Заказы — все заказы + назначение цены/исполнителя + отправка ссылки на оплату
  • Исполнители — CRM с кнопками связи (MAX/TG/тел/email)
  • Заявки — входящие лиды с лендинга
  • Каталог — управление листингами
  • Наценки — редактор ставок
  • Статистика — метрики

Потоки данных

Заказчик: Лендинг / /order/new → POST /api/orders → n8n уведомление админу → Админ назначает цену + исполнителя → Секретная ссылка в мессенджер → /my/{token} дашборд → Оплата → Подтверждение
Исполнитель: /join или Telegram/MAX бот → POST /api/contractors → n8n уведомление админу → Админ связывается через CRM → Назначает на заказ
Админ: /admin → Видит заказы + исполнителей → Назначает исполнителя на заказ → Выставляет display_price → Отправляет ссылку на оплату заказчику → Отправляет секретную ссылку на дашборд
